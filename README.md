# MyNBA
Live NBA scores and Latest News, Highlights, Engage with community and be a voice for your team.



## Technologies Used 

- MVC Architecture
- API
- Firebase
- LiveData
- Lottie Animation




## Features

- League news
- Games times and scores
- Game Status
- Game Box Score
- Game Highlights
- User comments on games
- League Standings
